React + Vite Setup Guide

Alright — let’s go step-by-step and get you a fresh React project running with Vite from scratch. We’ll use the modern way (fast, minimal, and no unnecessary installs).

1️⃣ Install Node.js

If you don’t already have Node.js:

  • Download from: https://nodejs.org
  • Install the LTS version.
  • After install, check:
node -v npm -v

2️⃣ Create a New React Project with Vite

In your terminal:

npm create vite@latest my-react-app

It will ask you:

  • Project name? → Press Enter or type your name.
  • Select a framework → Choose React (or React + TypeScript if you prefer TS).
  • Variant → Choose JavaScript or TypeScript.

3️⃣ Go into Your Project

cd my-react-app

4️⃣ Install Dependencies

npm install

5️⃣ Start the Development Server

npm run dev

It will show something like:

Local: http://localhost:5173/

Open that link in your browser.

6️⃣ Project Structure (Basic)

my-react-app/
  ├── public/        # static files
  ├── src/           # your React code
  │   ├── App.jsx    # main app component
  │   └── main.jsx   # entry point
  ├── package.json   # dependencies and scripts
  ├── vite.config.js # vite settings

7️⃣ To Stop the Server

Press Ctrl + C in the terminal.
