A quick reference for common Git commands, including initializing a repository, tracking changes, committing, and pushing to GitHub.

4Ô∏è‚É£ Restore Changes

  • Restore all changes: 
    git restore .

    // Reverts all changes in the working directory

  • Restore a specific file: 
    git restore MyFile.cs

    // Reverts a specific file to the last committed state

  • Restore multiple specific files: 
    git restore File1.cs File2.cs

    // Reverts multiple files to the last committed state

5Ô∏è‚É£ Add Changes (stage files for commit)

  • Add all files: 
    git add .

    // Stages all changes for commit

  • Add a specific file: 
    git add MyFile.cs

    // Stages a single file for commit

  • Add multiple specific files: 
    git add File1.cs File2.cs

    // Stages multiple files for commit

8Ô∏è‚É£ Return to Your Current Work (main branch)
